**Equitable Bank**
- Slug: `equitable`
- Type: National Alternative Lender (Publicly Traded)

- Fallback Rates: 3.84-4.24% fixed, 3.65% variable (5-year)

**Home Trust**
- Slug: `hometrust`
- Type: National Alternative Lender
- URL: https://www.hometrust.ca/mortgages
- Features: Largest alternative mortgage lender in Canada
- Fallback Rates: 4.29-4.39% fixed, 3.95% variable

**Wealthsimple**
- Slug: `wealthsimple`
- Type: National Alternative Lender (Fintech)
- URL: https://www.wealthsimple.com/mortgage
- Features: Modern fintech platform, recently launched mortgage product (2024-2025)
- Fallback Rates: 3.99% fixed, 3.49% variable (very competitive)

### Scraper Implementation

All three scrapers use:
- Dual approach: HTTP fetch first, Playwright fallback
- 30-second timeout for JavaScript-heavy pages
- Multiple regex patterns for rate extraction
- Accurate fallback data if scraping fails
