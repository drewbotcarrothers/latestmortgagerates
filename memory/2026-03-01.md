## Key Technical Discoveries

### Working Browser Profile
```python
browser(action="start", profile="openclaw")  # Port 18800
browser(action="open", targetUrl="...", profile="openclaw")
browser(action="snapshot", refs="aria", maxChars=15000)
```

- Terms captured: 6mo, 1yr, 2yr, 3yr, 4yr, 5yr, 7yr, 10yr
- Types: Fixed, Variable
